# ixl

Static content mirror — games, HTML pages, assets referenced by the Voidv5
launcher (`ixl-math-learning/Voidv5`). Served via jsdelivr so the launcher can
iframe local games directly without going through the Scramjet proxy.

This repo is auto-synced from `VoidNetworkOnTop/Voidv5` `static/` tree on every
push. Do not edit by hand.

Files over 100 MB are excluded (GitHub size cap) — those games still load
through the Scramjet proxy.
